# 🎬 AniMatch

AniMatch est une application Fullstack moderne permettant de découvrir et de naviguer dans le Top 100 des meilleurs animés. Le projet utilise l'API Jikan pour l'ingestion de données et offre une expérience utilisateur fluide grâce à une pagination optimisée.

## 🚀 Fonctionnalités

- **Dashboard Dynamique** : Affichage d'une grille 2x5 (10 animés par page).
- **Pagination Avancée** : Navigation par boutons (Suivant/Précédent) et saut direct à une page spécifique.
- **Base de Données Locale** : Ingestion des données depuis l'API Jikan vers PostgreSQL pour des performances accrues.
- **Design Responsive** : Interface adaptée aux mobiles et tablettes avec SCSS.
- **Dockerisé** : Environnement prêt à l'emploi avec Docker Compose.

## 🛠️ Stack Technique

- **Frontend** : Angular 18+, SCSS, HttpClient.
- **Backend** : Java 21, Spring Boot 3, Spring Data JPA.
- **Base de données** : PostgreSQL.
- **Conteneurisation** : Docker / Docker Compose.
